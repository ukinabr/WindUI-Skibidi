## Viewport Element

Viewport embeds a `ViewportFrame` and can optionally allow drag and pinch interaction.

<Tabs defaultValue="basic">
    <TabsList>
        <TabsTrigger value="basic">Basic</TabsTrigger>
        <TabsTrigger value="full">Full</TabsTrigger>
    </TabsList>
    <TabsContent value="basic">
```luau
local Part = Instance.new("Part")
Part.Size = Vector3.new(2, 2, 2)

Tab:Viewport({
    Object = Part,
})
```
    </TabsContent>
    <TabsContent value="full">
```luau
local Model = workspace:WaitForChild("PreviewModel"):Clone()

local Preview = Tab:Viewport({
    Object = Model,
    Height = 220,
    Interactive = true,
    Focused = true,
})

Preview:Focus()
```
    </TabsContent>
</Tabs>

## Parameters

<TypeTable
