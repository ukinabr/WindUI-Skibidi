---
title: Video
description: Render Roblox or cached webm video media in WindUI Modded
---

import TypeTable from '@/components/TypeTable';

`Video` renders a `VideoFrame` in the current tab or container.
HTTP `.webm` sources are cached into the window asset folder when the executor supports `writefile` and `getcustomasset`.

```luau
Tab:Video({
    Video = "video.webm",
    AspectRatio = "16:9",
    Radius = 18,
})
```

```luau
Tab:Video({
    Video = "https://example.com/preview.webm",
    AspectRatio = "16:9",
})
```
