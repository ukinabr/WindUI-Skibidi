## StatCard Element

StatCard is a metric block with a large value, subvalue and trend dot.

<Tabs defaultValue="basic">
    <TabsList>
        <TabsTrigger value="basic">Basic</TabsTrigger>
        <TabsTrigger value="full">Full</TabsTrigger>
    </TabsList>
    <TabsContent value="basic">
```luau
local FPS = Tab:StatCard({
    Title = "FPS",
    Value = "144",
    SubValue = "+12% from last run",
})
```
    </TabsContent>
    <TabsContent value="full">
```luau
local Stat = Tab:StatCard({
    Title = "Users",
    Desc = "Active sessions",
    Icon = "users",
    Value = "1,204",
    SubValue = "+18 today",
    Trend = "Up",
    TrendColor = "#16a34a",
    ValueTextSize = 28,
})

Stat:SetValue("1,320", "+116 today")
Stat:SetTrend("Up")
```
    </TabsContent>
</Tabs>
