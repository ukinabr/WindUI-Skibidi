# Tab Section
Tab sections are used to organize tabs into groups


<Tabs defaultValue="small">
    <TabsList>
        <TabsTrigger value="small">Basic</TabsTrigger>
        <TabsTrigger value="full">Full</TabsTrigger>
    </TabsList>
    <TabsContent value="small">
```luau
Window:Section({
    Title = "Tab Section example"
})
```
    </TabsContent>
    <TabsContent value="full">
```luau
Window:Section({
    Title = "Tab Section example", -- title. required
    Icon = "bird", -- icon. optional
    Opened = true, -- whether the section is opened. optional
})
```
    </TabsContent>
</Tabs>


## Tab Section Usage
```luau
local section = Window:Section({
    Title = "Tab Section example"
})

-- creating tabs in the section
section:Tab({ Title = "Tab 1" })
section:Tab({ Title = "Tab 2" })
section:Tab({ Title = "Tab 3" })
```
