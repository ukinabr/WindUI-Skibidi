`Card` is a layout element that can hold normal WindUI elements inside it.
It supports `HStack`, `VStack`, `Button`, `Toggle`, and the rest of the element API because it is a real nested container.

## Basic Card

```luau
local Card = Tab:Card({
    Title = "Card Navigation",
    Desc = "Cards can contain elements and actions.",
    Icon = "panels-top-left",
    Color = "#0EA5E9",
})

Card:Button({
    Title = "Normal element inside card",
    Icon = "mouse-pointer-click",
})

local Row = Card:HStack({
    MinChildWidth = 140,
})

Row:Badge({
    Title = "State",
    Value = "Ready",
    Variant = "Success",
})
```

## CardButton

```luau
Card:CardButton({
    Title = "Run action",
    Icon = "bell",
    Callback = function()
        print("card action")
    end,
})
```

## CardTab

`CardTab` creates a card action that opens a dedicated tab.
Pass `Build` to populate that tab immediately.

```luau
Card:CardTab({
    Title = "Open details",
    Icon = "arrow-right",
    TabTitle = "Card Detail",
    TabIcon = "panels-top-left",
    Build = function(DetailTab)
        DetailTab:Callout({
            Title = "CardTab Page",
            Desc = "This page was created from a card.",
            Variant = "Success",
        })
    end,
})
```
