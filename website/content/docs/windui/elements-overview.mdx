# Elements Overview

WindUI Modded now documents the element set by purpose instead of putting every control into one long mixed list.

![Elements preview](/windui/previews/windui-modded-preview.png)

## Core controls

| Element | Use it for |
| --- | --- |
| [Button](/docs/windui/button) | Run actions from a compact control. |
| [Toggle](/docs/windui/toggle) | Enable or disable a script feature. |
| [Slider](/docs/windui/slider) | Pick a numeric value quickly. |
| [Stepper](/docs/windui/stepper) | Mobile-friendly numeric changes with minus and plus controls. |
| [Input](/docs/windui/input) | Single-line text values. |
| [TextArea](/docs/windui/textarea) | Multi-line text input. |
| [Dropdown](/docs/windui/dropdown) | Pick from a list with fixed mobile positioning. |

## Selection groups

| Element | Use it for |
| --- | --- |
| [Segmented Control](/docs/windui/segmentedcontrol) | Switch between short modes. |
| [Radio Group](/docs/windui/radiogroup) | Choose one option from a visible set. |
| [Checkbox Group](/docs/windui/checkboxgroup) | Choose multiple options. |
| [Chip List](/docs/windui/chiplist) | Filter tags and small presets. |
| [Colorpicker](/docs/windui/colorpicker) | Pick colors for themes or features. |
| [Keybind](/docs/windui/keybind) | Bind keyboard and compact mobile actions. |

## Display and status

| Element | Use it for |
| --- | --- |
| [Paragraph](/docs/windui/paragraph) | Static text or media notes. |
| [Callout](/docs/windui/callout) | Important contextual messages. |
| [Badge](/docs/windui/badge) | Small labels and state chips. |
| [StatusCard](/docs/windui/statuscard) | Server, feature or runtime status. |
| [StatCard](/docs/windui/statcard) | Metrics such as FPS, ping or player count. |
| [KeyValue](/docs/windui/keyvalue) | Dense config summaries. |
| [ActionList](/docs/windui/actionlist) | Command rows with actions. |
| [MeterGroup](/docs/windui/metergroup) | Multiple progress values in one block. |
| [DiscordCard](/docs/windui/discordcard) | A dedicated Discord invite card with copy and join actions. |
| [Card](/docs/windui/card) | Visual page cards that open dedicated tabs, with optional nested content. |

## Layout and motion

| Element | Use it for |
| --- | --- |
| [Sections](/docs/windui/sections) | Organize content into collapsible groups. |
| [TabSection](/docs/windui/tabsection) | Split content inside a page. |
| [TabBox](/docs/windui/tabbox) | Multi-tab content inside a section box. |
| [Group](/docs/windui/group) | Visually combine related controls. |
| [HStack and VStack](/docs/windui/hstackvstack) | Build rows and columns inside examples or custom panels. |
| [Timeline](/docs/windui/timeline) | Show events and progress history. |
| [Accordion](/docs/windui/accordion) | Hide secondary details until needed. |
| [EmptyState](/docs/windui/emptystate) | Show an intentional blank state with action guidance. |
| [Path2D](/docs/windui/path2d) | Draw animated 2D paths. |

| [Space](/docs/windui/space) | Add deliberate spacing when needed. |
