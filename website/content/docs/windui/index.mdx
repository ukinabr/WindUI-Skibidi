# About WindUI Modded

WindUI Modded is a Roblox Luau UI library fork focused on mobile-friendly script interfaces, liquid glass visuals, richer elements and practical documentation.

The runtime API still uses the familiar `WindUI` variable in Luau examples. The docs and website use **WindUI Modded** so users can tell this fork apart from the original project.

## What this fork documents

- Liquid glass window styling, topbar controls and watermarked builds.
- Mobile-aware keybind menu, dropdown positioning and compact element sizing.
- New grouped elements such as DiscordCard, StatusCard, StatCard, KeyValue, ActionList, MeterGroup, Timeline, Accordion, EmptyState, Stepper, TextArea, ChipList, RadioGroup, CheckboxGroup and SegmentedControl.
- TabBox pages, Path2D previews, Section Box tabs and layout primitives.
