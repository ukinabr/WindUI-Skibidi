## Callout Element

Callout is a simple information block using variants and icons.

<Tabs defaultValue="basic">
    <TabsList>
        <TabsTrigger value="basic">Basic</TabsTrigger>
        <TabsTrigger value="full">Full</TabsTrigger>
    </TabsList>
    <TabsContent value="basic">
```luau
Tab:Callout({
    Title = "Heads up",
    Desc = "This action uses saved config.",
})
```
    </TabsContent>
    <TabsContent value="full">
```luau
Tab:Callout({
    Title = "Build Ready",
    Desc = "Everything compiled successfully.",
    Variant = "Success",
    Icon = "circle-check",
    Color = Color3.fromHex("#16a34a"),
    Size = UDim2.new(1, 0, 0, 72),
})
```
    </TabsContent>
</Tabs>
